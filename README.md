# Project Description
 - In this project I have compiled and fit a CNN model that recognizes images from the following labels
    - T-shirt/top
    - Trouser
    - Pullover
    - Dress
    - Coat
    - Sandal
    - Shirt
    - Sneaker
    - Bag
    - Ankle boot
# Libraries Used
 - Tensorflow Keras
 - Matplotlib
 - Numpy
 - Pandas
